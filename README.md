# Travel

# アプリケーションの目的
このアプリケーションはコロナ渦の中旅行にはいきたいが<br>
人混みは避けたいと思う人を対象に作成した。<br>
旅行先を調べても有名なとこなどしか基本的に出てこないため、<br>
人混みを避けることができない。<br>
そこで、穴場のスポットなどあまり知られてないような場所などの<br>
お勧めのスポットを共有することによって、<br>
旅行する際などの参考にすることができる。

# 利用方法
お勧めのスポットを投稿して共有する<br>
お気に入り登録をする<br>
他者の投稿に対してコメントをする<br>

# 実装機能
1.ユーザー管理機能<br>
2.投稿機能<br>
3.編集削除機能<br>
4.お気に入り機能<br>
5.コメント機能<br>
